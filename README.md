Este é meu primeiro projeto de site que desenvolvi utilizando habilidades adquiridas no meu curso Técnico de Informática. O trabalho foi focado na aplicação prática de HTML5 semântico para criar uma estrutura organizada e eficiente. O tema escolhido foi a psicologia das cores no marketing, onde explorei como a comunicação visual pode ser estruturada através do código.

Durante o desenvolvimento, foquei na organização de múltiplos arquivos e na criação de uma navegação funcional entre as páginas. Além da escrita do código, este projeto marcou meu primeiro contato com o controle de versão, passando por todas as etapas de instalação do Git e a publicação final no GitHub. Este projeto serve como base para consolidar meus conhecimentos em estruturação de dados e marcação de texto, preparando o caminho para estudos mais avançados em desenvolvimento web.
